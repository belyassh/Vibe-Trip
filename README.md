# Vibe-Trip

Адаптивный шаблон web-приложения экскурсионного агентства с интерактивными витринами экскурсий.

## Что реализовано

- Каталог экскурсий из JSON (без хардкода в HTML).
- Поиск и фильтрация по направлениям.
- Детальная карточка экскурсии в модальном окне:
- Слайдер фото
- Название
- Обзор
- Полное описание
- Стоимость
- Что включено
- Что взять с собой
- Форма заявки:
- Имя
- Фамилия
- Количество человек
- Итоговая стоимость (автоматический расчет)
- Название отеля или точка Google Maps
- Желаемая дата
- Телефон или ник Telegram
- Отправка заявки на email через сервис формы (например, Formspree).
- Резервная адресация в Telegram, если email-сервис не настроен или недоступен.

## Структура проекта

- `index.html` - каркас интерфейса и секций.
- `styles/main.css` - визуальный стиль, адаптивность, анимации.
- `scripts/app.js` - загрузка данных, рендер витрин, слайдер, фильтры, форма.
- `data/excursions.json` - конфигурация агентства, Telegram и список экскурсий.

## Как управлять экскурсиями

Все изменения выполняются в `data/excursions.json`.

### Добавить экскурсию

1. Создайте новый объект в массиве `excursions`.
2. Заполните поля:
	- `id` (уникальный)
	- `title`
	- `overview`
	- `description`
	- `price`
	- `included` (массив)
	- `bring` (массив)
	- `tags` (массив)
	- `images` (массив URL)

### Изменить экскурсию

Отредактируйте соответствующий объект по `id`.

### Удалить экскурсию

Удалите объект из массива `excursions`.

## Отправка заявок на email

В `data/excursions.json` настройте:

- `emailService.endpoint` - endpoint формы из сервиса (например, `https://formspree.io/f/your_form_id`).

После этого заявка будет отправляться напрямую из формы на email.

## Telegram (fallback)

В `data/excursions.json` настройте:

- `telegram.managerUsername` - username менеджера без `@`.

Если email-сервис не настроен или не отвечает, форма откроет Telegram с готовым текстом заявки и ссылкой на менеджера.

## Запуск

Откройте `index.html` в браузере.

Для корректной загрузки JSON рекомендуется запускать проект через локальный web server.
